# FormMasks

Masks for forms using Regexp and Javascript in Brazilian format.

- **CPF**
- **RG**
- **CEP**
- **Tel**
- **Cel**
- **Date** *coming soon*
- **Hour** *coming soon*

*[Do a test](http://clarabatt.github.io/form-masks/)*
